Receipt Splitter

Scrape Sam's Club receipts and export item splits to CSV or Google Sheets.

Install (developer)

  1. Clone or download this repo to your machine.
  2. Open Chrome → `chrome://extensions` → enable Developer mode → "Load unpacked".
  3. Select the `extension/` folder from this repo.
  4. Open a Sam's Club order/cart page and click the extension icon. Click "Scrape" then export.

Export options: CSV (no auth), or append to Google Sheets (requires OAuth setup — see below).

Google Sheets setup

  1. Create a Google Cloud Project and enable the Google Sheets API.
  2. Configure OAuth consent screen (External or Internal) and create OAuth 2.0 Client ID.
  3. If using `chrome.identity.getAuthToken`, you don't need to add redirect URIs if you use the built-in flow — follow Chrome extension OAuth docs.
  4. Copy your spreadsheet ID into extension/background.js replacing YOUR_SPREADSHEET_ID.
  5. In the extension, the first Sheets export will ask you to sign in and allow scopes.
